# h-share
a share plugin

## Quick start

```javascript
import share from 'happy-share';
share({
    title: 'my title',
    summary: 'my summary',
    pic: 'http://qzonestyle.gtimg.cn/aoi/sola/20150617094556_OvfOpoRKRB.png',
    url: location.href
})
```
